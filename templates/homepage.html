Information

  Rivia is a automated Deep Learning based Question Answering system. It will answer your question on the data (table or text) you provide.

Instructions

We've three demos for you

  1. Question Answering on Text and Table

    Just check this QA system on both Text and Table. Upload a CSV or TXT for table and text respectively (We'll add support for other file extensions like PDF and DOCX). If you don't have any data you can choose the default data from the drop down menu.

    Now, write or speak the question by providing the input on the right pane.

  2. Automatic Table fetch from Wikipedia

    This is a demonstration for QA systems on Wikipedia tables. For now we have kept the table fixed. Link to table. This will fetch latest COVID-19 cases by country and you can ask questions related to that. For instance, "How many countries have at least 5000 deaths?" or "What are the active cases in Canada right now?" etc.

  3. Automatic Article fetching from Wikipedia

    In this demo you'll see a text box where you can insert topics and the system will fetch its introductory paragraph (We'll provide entire wikipedia QA soon). You can try inserting topics like "Conversational AI", "covid-19", "Sachin Tendulkar" or "India". In some cases you won't see any data return because our system might be unable to fetch that page.

    Ask questions from the passage which you can see on the left pane.
